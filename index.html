☁️ 需要配置 Supabase 才能使用云端同步
1
访问 supabase.com，用 GitHub 或 Google 账号登录

点右上角「Start your project」

2
创建新项目，Region 选「Southeast Asia（Singapore）」

新加坡节点，国内访问速度最快

3
进入项目 → Settings → API，复制两个值

Project URL 和 anon public key

4
在 SQL Editor 运行建表语句（教程里有）
5
把两个值填入 index.html 顶部，重新部署
📋 详细图文教程见「Supabase配置教程.html」
